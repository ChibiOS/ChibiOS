SB Host Switched

Host and Sandbox Layout

The privileged host builds the filesystem view. Its VFS root is a ROMFS image containing the web content, with a streams driver mounted on /dev.

Sandbox 2 receives that VFS root through sbSetFileSystem(). Inside the sandbox, the POSIX syscalls map open, read, fstat and directory operations onto the host VFS.

The HTTP server itself remains unaware of the host-side VFS details. It just uses the POSIX binding hooks required by lwIP HTTPD custom files.
