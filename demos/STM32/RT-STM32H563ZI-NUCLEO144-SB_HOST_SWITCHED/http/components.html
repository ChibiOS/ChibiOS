SB Host Switched

Used Components

  • ChibiOS/RT as the kernel on the privileged host side.
  • Sandbox manager with virtual Ethernet and GPIO for sandbox 2.
  • lwIP HTTPD running inside the sandboxed RT-XHAL client.
  • Host-side ROMFS plus overlay VFS to provide files and /dev.
  • POSIX HTTPD binding that reads files through the sandbox syscalls.
