ChibiOS Sandbox Demo

SB Host Switched + lwIP HTTP

A sandboxed lwIP HTTP server serving files from a host-managed ROMFS.

What is running

Sandbox 2 owns the Ethernet virtual I/O and runs the lwIP HTTP server as an unprivileged application.

Where files come from

The privileged host instantiates a ROMFS image as the sandbox root filesystem, then overlays /dev for standard descriptors.

Why POSIX matters

HTTPD reads pages through open, read, fstat and close using the sandbox POSIX API.
