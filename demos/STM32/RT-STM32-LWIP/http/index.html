ChibiOS Demo

RT-STM32-LWIP

A compact HTTP demo built on ChibiOS/RT, the STM32 HAL and the lwIP TCP/IP stack.

What it shows

This application boots the RT kernel, initializes the Ethernet interface, starts lwIP and serves a tiny website over HTTP.

Why ROMFS fits

The website assets can live in flash as a read-only image, with no SD card, no FAT image and no generated lwIP fsdata table.

What is wired now

The generated ROMFS image is instantiated as the VFS root and served by lwIP HTTPD through the custom VFS binding.
