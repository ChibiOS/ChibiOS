ROMFS Notes

The files in this directory are plain source assets. They are converted into a ChibiOS ROMFS image and emitted as C sources under ./source.

That makes the web content easy to version, diff and regenerate while keeping the final runtime representation read-only and self-contained.

In this demo the generated ROMFS is instantiated directly as the VFS root, and the HTTP server serves these files through the dedicated VFS custom-file binding.
