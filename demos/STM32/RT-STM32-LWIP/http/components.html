RT-STM32-LWIP

Used Components

  • ChibiOS/RT for scheduling, timing and synchronization.
  • STM32 HAL/XHAL for board bring-up and Ethernet peripheral support.
  • lwIP for IPv4 networking and HTTPD.
  • Port-specific board setup from portab.c.
  • Generated ROMFS content for static pages stored in flash.
