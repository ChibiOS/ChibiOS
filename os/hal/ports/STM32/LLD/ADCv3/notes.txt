STM32 ADCv3 driver.

Driver capability:

- Supports the STM32 "fast" ADC found on F3, L4, L4+, G4 and WB sub-families.

The file registry must export:

STM32_HAS_ADCx                  - ADCx presence flag (1..5).
STM32_ADC1_HANDLER              - IRQ vector name for ADC1.
STM32_ADC1_NUMBER               - IRQ vector number for ADC1.
STM32_ADC2_HANDLER              - IRQ vector name for ADC2.
STM32_ADC2_NUMBER               - IRQ vector number for ADC2.
STM32_ADC3_HANDLER              - IRQ vector name for ADC3.
STM32_ADC3_NUMBER               - IRQ vector number for ADC3.
STM32_ADC4_HANDLER              - IRQ vector name for ADC4.
STM32_ADC4_NUMBER               - IRQ vector number for ADC4.
STM32_ADC5_HANDLER              - IRQ vector name for ADC5.
STM32_ADC5_NUMBER               - IRQ vector number for ADC5.

If there is no DMAMUX then the file registry must export also:

STM32_ADCx_DMA_MSK              - Mask of the compatible DMA channels (1..4).
STM32_ADCx_DMA_CHN              - Mask of the channels mapping (1..4).

STM32G4 errata notes:

- ES0430, section 2.7.10 "ADC channel 0 converted instead of the required ADC
  channel", affects the first regular or injected conversion following DMA
  end-of-transfer after a regular sequence read by DMA in one-shot mode. The
  affected conversion may sample the internal channel 0 and return zero instead
  of the requested channel.
- This can be observed with repeated non-circular adcConvert() operations: the
  first sequence is valid, subsequent sequences can have the first rank equal
  to zero even though the ADC and DMA setup registers are programmed
  identically before ADSTART.
- Do not hide this in the driver with an implicit dummy conversion. A dummy
  conversion changes timing and ADC bandwidth, and starting then immediately
  stopping a dummy conversion does not consume the erratum path. Applications
  and demos using repeated DMA one-shot conversions on affected devices must
  explicitly account for the limitation, for example by performing and
  discarding a completed dummy conversion, using a continuous/circular
  conversion pattern, or selecting another timing strategy.
